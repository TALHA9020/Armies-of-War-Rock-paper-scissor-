# Armies of War ⚔️

**Armies of War** ایک اسٹریٹیجک اینڈرائیڈ گیم ہے جہاں آپ اپنی فوج کی قیادت کرتے ہیں اور عالمی تسلط کے لیے مختلف اتحاد (Alliances) بناتے ہیں۔ یہ گیم "Rock-Paper-Scissors" کے کلاسک اصولوں کو ایک بڑے جنگی نقشے پر منتقل کرتی ہے۔

## نمایاں خصوصیات (Features) 🚀

* **کثیر الافوائجی نظام (Multi-Army System):** پورے نقشے پر 2 سے لے کر 10 تک مختلف افواج کے درمیان مقابلہ۔
* **اتحاد اور سیاست (Alliances):** آپ کے پاس اختیار ہے کہ آپ اکیلے لڑیں یا دوسری افواج کے ساتھ مل کر ایک طاقتور اتحاد بنائیں۔
* **رنگوں کا انتخاب:** ہر فوج کی اپنی منفرد پہچان اور رنگ۔
* **ٹرن بیسڈ گیم پلے:** باری کے حساب سے یونٹس بھیجنے اور حملہ کرنے کا نظام۔
* **یونٹ واپسی کا رول:** جنگ جیتنے کے بعد آپ کی بچی ہوئی فوج واپس آپ کے ہیڈ کوارٹر میں شامل ہو جاتی ہے۔

## گیم کے اصول (How to Play) 🎮

یہ گیم **RPS (Rock-Paper-Scissors)** کے اصولوں پر کام کرتی ہے:
1.  **Rock (چٹان):** Scissors کو ہراتا ہے۔
2.  **Paper (کاغذ):** Rock کو ہراتا ہے۔
3.  **Scissors (قینچی):** Paper کو ہراتا ہے۔

## تکنیکی معلومات (Technical Details) 🛠️

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose
* **Architecture:** StateFlow based Battle Engine
* **Build System:** Gradle 8.5

## انسٹالیشن (Installation) 📦

1.  اس ریپوزٹری کو کلون کریں۔
2.  Android Studio میں پروجیکٹ اوپن کریں۔
3.  `./gradlew assembleDebug` کے ذریعے APK بنائیں۔
